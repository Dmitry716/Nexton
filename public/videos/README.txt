Hero-слайдер (главная страница)
==============================

Поместите сюда видео для фона hero-секции.

Имя файла: hero-bugatti.mp4

Содержание ролика:
- Девушка подходит на шпильках
- Садится в Bugatti Chiron
- Уезжает в закат

Формат: MP4 (рекомендуется H.264). Разрешение: минимум 1920×1080 для качества на больших экранах.

Если файла hero-bugatti.mp4 нет, фон hero будет пустым — добавьте свой ролик или временно скопируйте сюда любой MP4 с именем hero-bugatti.mp4.
